1. 스트림(Stream)
	- 데이터의 입출력 경로
	- 프로그램 기준으로 입력, 출력 결정 (기본은 단반향)
	- 종류
		- 바이트(Byte)
			-바이트 단위 데이터 전송
			- 이미지, 멀티미디어
			
			- InputStream
				- int read() : 바이트 자체 입력받음
				
				
			- OutputStream
				- void write()
					
		- 문자(Character)
			- 문자에 특화
			- Reader, Writer
		
		
2. 직렬화(Serialization)
	- 인스턴스의 상태를 그대로 파일에 저장하거나 혹은 네트워크로 전송하고 다시 복원하는 방식
	- 자바에서 보조 스트림을 활용
		- ObjectOutputStream / ObjectInputStream
	
	- 클래스 변수, 메소드 포함되지 않음
	- 고려사항
		- 수정에 대한 부분에서 한계
		- 테스트 부담 증가 가능
		- 상속용으로 설계된 클래스의 경우는 직렬화를 하지 않는 쪽으로 고려 해봐야 함
