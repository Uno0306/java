  1. 예외(Exception)
  	- 사용자 혹은 개발자의 잘못으로 발생하는 오류, 처리 가능한 에러
  	- 종류
  		- 일반 에외 : 컴파일 체크 계열(Class, ...)
  			- java.lang.RuntimeException을 제외한 모든 예외 클래스
  		
  		- 실행 예외
  			- java.lang.RuntimeException 상속받는 모든 예외 클래스
  	
  	- 처리
  		try{
  			예외 발생 가능한 수행 코드;
  		} catch(예외 객체){
  			예외 처리 수행 코드;
  		} finally{
  			예외 발생 여부 무관, 무조건 실행되는 코드 블럭
			** 자원 반환 로직
			(ex. DB 접속 해제, …)
  		}
